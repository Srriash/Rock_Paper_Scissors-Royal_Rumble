# Rock Paper Scissors Royal Rumble

Retro, arcade-style Rock-Paper-Scissors game built in Python.

Running:

1. Install dependencies (use your Python environment):

```powershell
python -m pip install -r requirements.txt
```

2. Run the Pygame GUI:

```powershell
python rps_pygame.py
```

Controls:
- Click a choice (ROCK, PAPER, SCISSORS) to play a round.
- Press `ESC` to quit.

Notes:
- The game uses the existing `rps_logic.py` for the game rules.
- No external assets required.
